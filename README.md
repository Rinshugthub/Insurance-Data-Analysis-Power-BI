# Insurance Data Analysis Dashboard – Power BI

## Project Overview
This project presents an interactive Power BI dashboard developed to analyze insurance data and generate business insights through KPIs and visual analytics.

The dashboard helps in:
- Tracking key insurance performance indicators (KPIs)
- Monitoring customer and policy trends
- Identifying business growth opportunities
- Improving data-driven decision making

## Tools & Technologies Used
- Power BI Desktop
- Data Cleaning
- Data Modeling
- DAX
- Data Visualization

## Key Insights
- Performance tracking using KPI cards
- Customer and policy analysis
- Interactive filtering and drill-down
- Trend and comparison analysis
